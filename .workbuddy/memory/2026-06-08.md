## 修复 SIM 切换器运营商名显示为 Unknown（2026-06-08）

### 问题
- 信号圆圈 ✅ 正确显示运营商名（如"中国移动"）
- 顶部 SIM 切换器 ❌ 始终显示 "Unknown"
- 同一个数据源，两个显示位置不一致

### 根因
**CellularPage.kt:122** `simOptions` 使用 `remember {}`（无 key），只在初次组合时读取一次值。此时数据流还没启动，`_sim1Data.value` 为 `null`，所以永远缓存了 "Unknown"。

信号圆圈使用 `by viewModel.currentSignalData`（State 委托），数据到达时自动重组，所以能正确显示。

### 修复
- `remember {}` → `by remember { derivedStateOf { ... } }`
- `derivedStateOf` 会响应式跟踪内部读取的 State（`_sim1Data` / `_sim2Data`），数据到达后自动重新计算
- 添加了 `derivedStateOf` import
