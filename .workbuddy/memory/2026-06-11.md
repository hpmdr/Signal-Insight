## 18:29 — 用户指正：5G NR 底层协议有 RSSI，只是 Google 未开放 API
- **用户的深刻见解**：说"5G 没有 RSSI"是片面的。NR 协议层 RSSI 确实存在，它是 SS-RSRQ 计算公式的分母（RSRQ = N × RSRP / RSSI）。Google 不公开接口是因为 TDD 动态波束使总功率（RSSI）波动太剧烈，参考价值低，避免误导开发者。
- 更新了 `metric_net_rssi` 中英文说明字符串，准确描述 5G 的 RSSI 状况
- 更新 MEMORY.md 参数表：5G NR 的 RSSI 列改为"❌ 无 API" + 协议层存在的备注
- **反思**：不要仅凭公开 API 有无接口就断定底层不存在，协议标准比 API 抽象层更底层。
